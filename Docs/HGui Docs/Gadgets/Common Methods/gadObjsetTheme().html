gadObj:setTheme()

gadObj:setTheme(theme, refreshWindow)

Setup the a custom gadget theme unlinking the gadget from the global theme, the current skin will be marked as custom skin.

INPUT

  • theme : A table with the gadget theme definition.
  • refreshWindow : True to reresh the window's contents at the end of the process.

OUTPUT

NOTES

This method is experimental and unfinished.
