HGui.gadgetFindByHWIdAndPos()

gadObj = HGui.gadgetFindByHWIdAndPos(HWId, x, y)

Check if the specified point into the given window pecifing its Hollywood ID HWId it's inside a gadget.
Returns the gadget if was found otherwise returns FALSE.

INPUT

  • HWId : Hollywood ID of the window we are looking for the gadget
  • x : Horizontal position
  • y : Vertical position

OUTPUT

  • gadObj : The found gadget object or FALSE if there is no gadgets at the given position.

NOTES

EXAMPLES 

 ; Check if there is gadget at (100, 100) on window 1
 Local result, g = HGui.gadgetFindByHWIdAndPos(1, 100, 100)
 If result
   DebugPrint("FOUND > Gadget name is : " .. g.name)
 EndIf
