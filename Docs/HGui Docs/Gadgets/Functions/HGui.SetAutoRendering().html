HGui.SetAutoRendering()

result = HGui.SetAutoRendering(value)

This function is used to switch the automatic rendering on or off.
It's usefull especially during the interface creation, when you are creating each single gadgets and their hierarchy, to avoid useless redrawing that can bring only flickering and slowdown effects.
 

INPUT

  • value : TRUE to enable the auto rendering, FALSE to deactivate it.

OUTPUT

  • result : TRUE if the operation was completed..
