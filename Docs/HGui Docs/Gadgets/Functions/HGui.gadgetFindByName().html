HGui.gadgetFindByName()

gadObj, index = HGui.gadgetFindByName(name)

Search the gadget object using its name, if found this function will return the gadget object and the position index in the HGui.Gadgets table. 
If the gadget isn't found False will be  returned.

INPUT

  • name : Gadget name to search for

OUTPUT

  • gadObj : The found gadget object or FALSE if there is no gadgets at the given position.
  • index : The gadget index in HGui.Gadgets table.

NOTES

EXAMPLES 

 ; Search for a gadget named 'myButton'
 Local result = HGui.gadgetFindByName("myButton")
 
 ; Check the result type because it can return FALSE or a gadget object
 If GetType(result) = #TABLE
   DebugPrint("FOUND > Gadget Name is : " .. result.Name)
 EndIf
