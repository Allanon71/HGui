Overview

OVERVIEW

Gadgets are graphical entities (with an optional skin) that you can use to reate GUIs or user interfaces, they are divided into two groups: passive and active.

Passive gadgets are graphical area that don't produce any event and you cannot attach actions to them, for example a BOX gadget or a LABEL gadget are both passive.

Active gadgets, instead, are gadgets that can produce events and you can interact with them in some way (mostly with your mouse). You can attach actions to them and they have one or more event that is triggered by an user action, like, for example, a mouse click.
