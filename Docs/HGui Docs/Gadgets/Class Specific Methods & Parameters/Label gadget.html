Label gadget

For common methods have a look here,

 

LABEL GADGET METHODS

  • gadObj:align(textAlignment, refresh)
    Changes the label's text alignment.
    - textAlignment : alignment constant(a)
    - refresh : True to refresh the gadget
     
  • gadObj:appendLine(content, autoScroll, refresh)
    Adds a line of text and optionally scroll to the latest added line.
    - content : text to add at the bottom of the current contents
    - autoScroll : True to auto scroll to the bottom
    - refresh : True to refresh the gadget
     
  • gadObj:clear(refresh)
    Clears the label's contents.
    - refresh : True to refresh the gadget
     
  • gadObj:fontMargins(margins, refresh)
    Sets the text margins (top, bottom, left, right).
    - margins : Table with the margins to set
    - refresh : True to refresh the gadget
     
  • gadObj:fontName(name, refresh)
    Changes the current font.
    - name : font name to set or a font constants for the inbuilt fonts
    - refresh : True to refresh the gadget
     
  • gadObj:fontSize(size, refresh)
    Changes the font size.
    - size : Font size to set
    - refresh : True to refresh the gadget
     
  • gadObj:fontStyle(style, refresh)
    Changes the font style.
    - style : Text style combination to set
    - refresh : True to refresh the gadget
     
  • cont_table, cont_string = gadObj:get()
    Returns the label's contents as a table of lines and as a whole string.
     
  • gadObj:setCaption(caption, refresh)
    Changes the label's contents.
    - caption : Text to set in the label (a stirng or a table of strings)
    - refresh : True to refresh the gadget
     
  • gadObj:setColor(color, refresh)
    Changes the text color.
    - color : color to set
    - refresh : True to refresh the gadget
     
  • gadObj:setWordwrap(enabled, refresh)
    Enables/disables the text wordwrap.
    - enabled : True to enable wordwrap
    - refresh : True to refresh the gadget
     
  • gadObj:scrollDown(rows, refresh)
    Scrolls down the text.
    - rows : Number of rows to scroll down
    - refresh : True to refresh the gadget
     
  • gadObj:scrollBottom(maximize, refresh)
    Scrolls to the last line of text.
    - maximize : True to maximize the visible text
    - refresh : True to refresh the gadget
     
  • gadObj:scrollTop(refresh)
    Scrolls to the first line of text.
    - refresh : True to refresh the gadget
     
  • gadObj:scrollUp(rows, refresh)
    Scrolls up the text.
    - refresh : True to refresh the gadget
     
  • gadObj:scrollTo(row, refresh)
    Scrolls to the given row.
    - row : scroll to the specified row
    - refresh : True to refresh the gadget
