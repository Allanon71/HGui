Additional arguments accepted by :New() method

  • Orientation : Defines the scrollbar orientation : #VERTICAL, #HORIZONTAL
  • Caption : Cursor text with support for the following tags:
      {value} : Replaced with the current value
      {value.##} : Replaced with the current value with two decimal places
      {intvalue} : Replaced with the current integer value
      {rangestart} : Replaced with the range start value
      {rangeend} : Replaced with the range end value
  • Range : A table to define the values range â†’ { Minimum, Maximum, Visible }
  • Value : Scrollbar initial value
  • Outer_Box : Standard options for the outer box (which is a Button)
  • Cursor : Standard options for the cursor box (which is a Button)
  • TextAngle : Caption rendering angle

 

Additional actions accepted by :New() method

  • OnChange : Triggered when scrollbar has changed its value
