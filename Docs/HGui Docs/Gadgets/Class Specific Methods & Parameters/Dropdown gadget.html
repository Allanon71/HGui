DROPDOWN GADGET METHODS

  • gadObj:selectItem(index, refresh)
    Select the item with the specified index
    - index : Index of the item to select
    - refresh : True to refresh the gadget

 

Additional arguments accepted by :New() method

  • value : Initial value. It can be a string, a table of string or the index of the items list
  • column : Column associtaed with the control
  • data : Items list data format:
    • ColumnSizes : A table with each column size in pixels
    • Headers : A table with the column headers
    • Items : A table with the items to show in the gadget
    • Selected : A table with the selected items
    • SortOrder : A table specifing the sort order for each columns

 

Additional actions accepted by :New() method

  • OnSelect : Triggered when an item has been select from the list
