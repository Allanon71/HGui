TEXT GADGET METHODS

  • result = gadObj:getText()
    Returns the current text gadget contents
     
  • gadObj:setText(value)
    Sets the current text gadget contents
    - value : Text to put in the text gadget

 

Additional arguments accepted by :New() method

  • gfx : A table to define the text box look (see theme definition), most common options are:
    • bgColor : Backgroud color
    • fgColor : Foreground color
    • bgCursor : Cursor's background color
    • fgCursor : Cursor's foreground color
    • adjustFontSize : True to automatically adjust the font size to the gadget size
  • allowNewLine : True to allow new line/multiline box
  • text : Initial text contents
  • align : Text alignment (#LEFT, #RIGHT, #CENTER), default is #LEFT
  • any other options usuported by the TP Object (NEED MORE DOCS)

 

Additional actions accepted by :New() method

  • OnInputEnd : Triggered when the gadget has lost its focus.
