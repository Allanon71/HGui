Additional arguments accepted by :New() method

  • VirtualSize : A table with the w and h fields to specify the virtual area size
  • VirtualOffset : A table with the x and y fields to specify the virtual area offsets
  • Container : A table with container customized options NEED MORE DOCS
  • VirtualBox : A table with virtual box customized options NEED MORE DOCS
  • Scrollers : A table with scrollers customized options NEED MORE DOCS
  • ScrollerSize : A table with scroller's sizes NEED MORE DOCS
  • HScrollerOFF : True to switch off the horizontal scroller
  • VScrollerOFF : True to switch off the vertical scroller
  • HAutoSense : True  to switch on the horizontal auto-sensing, this will switch off the horizontal scroller
  • VAutoSense : True  to switch on the vertical auto-sensing, this will switch off the vertical scroller
