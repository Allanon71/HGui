CANVAS GADGET METHODS

  • layerId = gadObj:addLayer(layerArgs)
    Add a new layer to the canvass and returns its id.
    - layerArgs : Optional table to customize the layer we are going to add, all arguments supported by CreateBrush() are supported.
    MORE DOCS NEEDED 
     
  • gadObj:removeLayer(layerId)
    Removes the specified layer from the canvass.
    - layerId : The layer id to remove
     
  • layers = gadObj:getLayers()
    Returns a list of layers, each entry points to the brush used to store the layer.
     
  • gadObj:refresh()
    Used to refresh the canvass gadget redrawing all the layers.
     
  • gadObj:beginDraw(layerId, mode)
    Activate the draw to the canvass, every rendering operation will be redirected to the specified layer until a call to :endDraw() is done.
    - layerId : The layer id where we want to draw
    - mode : The drawing combo mode to use
     
  • gadObj:scaleLayers(w, h, smooth)
    Scales all the layers to the specified size, optionally using 
    - layerId : The layer id where we want to draw
    - mode : The drawing combo mode to use
     
  • gadObj:resizeLayers()
    Resizes the layers to the current canvass size.

Additional arguments accepted by :New() method

  • CanvasSize : A table with the w and h fields to specify the canvass size
  • CanvasOffset : A table with the x and y fields to specify the canvass offsets
  • DefaultLayerArgs : A table with the additional layer argument (all brush creation arguments are supported)

 

Additional supported actions 

  • OnMove : Callback raised by the mouse movement within the canvass area
