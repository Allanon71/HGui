Additional arguments accepted by :New() method

  • source : Source image file or a preloaded brush id
  • mode : Determines the resize mode and can be:
    1 : Fit min, 2 : Fit max, 3 : Scale, 4 : Stretch
  • fadeIn : Set to True to have a fade in effects when the image is displayed
  • scale : A table to scale the source image
    • w : width
    • h : height
  • smooth : Set to True to smooth the image when resized
  • offset : A table to change the display offset of the image
    • x : Horizontal offset
    • y : Vertical offset
  • alpha : a value to set the alpha channel or #NONE
  • align : Hollywood constant to set the image alignment
  • borderColor : color of the borders generated when resizing
  • borders : A table to set the image borders in pixels
    • top : Top border
    • bottom : Bottom border
    • left : Left border
    • right : Right border
