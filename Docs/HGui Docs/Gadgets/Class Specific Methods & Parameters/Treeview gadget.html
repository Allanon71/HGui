TREEVIEW GADGET METHODS

  • gadObj:SelectAll(Refresh)
    Selects all list items (only in listview with multiselect enabled)
    - refresh : Set to True to refresh the gadget
     
  • gadObj:UnselectAll(Refresh) / gadObj:DeselectAll(refresh)
    Deselects all list items
    - refresh : Set to True to refresh the gadget
     
  • gadObj:openNodes(refresh)
    Open all the tree nodes
    - refresh : Set to True to refresh the gadget
     
  • gadObj:closeNodes(refresh)
    Close all the tree nodes
    - refresh : Set to True to refresh the gadget
     
  • gadObj:remove(index, refresh)
    Removes the specified item at the given index
    - index : item's index to remove
    - refresh : Set to True to refresh the gadget
     
  • gadObj:removeAll(refresh)
    Removes all the list items
    - refresh : Set to True to refresh the gadget
     
  • gadObj:removeSelected(refresh)
    Removes selected items from the list
    - refresh : Set to True to refresh the gadget
     
  • itemsList, selectedList = gadObj:get()
    Retrieves the item list and the selected list
     
  • gadObj:insert(record, index, selected, refresh)
    Scales all the layers to the specified size, optionally using 
    - record : The item record to insert
    - index : The position where to insert the item, if nil the item is added at the bottom of the list
    - selected : Set to True to select this item
    - refresh : Set to True to refresh the gadget
     
  • gadObj:setItems(itemsList, refresh)
    Replaces the current list with the one specified
    - itemsList : The new list data set
    - refresh : Set to True to refresh the gadget
     
  • gadObj:rebuildLayout()
    Rebuild and refresh the entire gadget
     

Additional arguments accepted by :New() method

  • Data : A table used to specify the gadget contents, it should have the following fields:
    • selected : A table used to specify the selected items (True = selected)
    • alignments : A table used to specify for each column the alignment (using HGui constants)
    • columnSizes : A table to specify each column size in pixels
    • headers : A table to specify each column title/header
    • items : A table used to pass the data to show in the list, each record has the following additional fields::
      • _childs : A table used to define all this node's childs
      • _selected : A table to set all selected childs
      • _opened : A table to set if the childs are opened nodes or not
    • sortOrder : A table used to specify for each column the sorting order
    • icons : A table used to specify for each record an optional icon
    • columnHandlers : A table to specify for each column a conversion function to display converted/formatted data instead of the raw data.
  • OuterBox_Borders : Distance between main container and its internal gadgets
  • VScrollerGap : Distance between contents area and the vertical scroller
  • ScrollerSize : Scrollers size in pixels (override the theme)
  • ColumnHeight : Height of the column headers (override the theme)
  • VerticalGap : Distance between all vertical items (top, body, horizontal scroller)
  • Vertical_Borders : Distance between all internal items (top, body, horizontal scroller)
  • ColumnsGap : Distance between column headers (override theme)
  • ColumnsBorders :Distance between column headers borders
  • Borders : Gadget's borders
  • VLines : Vertical lines between the columns True or False
  • VLinesColor1 : Most left line color
  • VLinesColor2 : Most right line color
  • Multiselect : Allow multiple row selection True or False
  • AutoSelectChilds : Auto-select item's childs when selecting a parent node
  • AutoUnselectChilds : Auto-unselect item's childs when unselecting a parent node
  • FontName : Font name
  • FontSizeOffset : Adjust font size, usefull for multiline cells
  • FontStyle : Font style
  • FontColor : Font color (unselected)
  • EntryHeight : Height in pixels of a single row & base font size
  • AlternateLines : Alternate lines background color (True or False)
  • HeadersOFF : True to hide column headers
  • HScrollerOFF : True to hide the horizontal scroller
  • VScrollerOFF : True to hide the vertical scroller
  • WheelPageQualifier : Override default page key qualifier (LSHIFT)
  • WheelSEQualifier : Override default start/end key qualifier (LCTRL)
  • IconsSizeOffset : +/- size offset
  • IconsXOffset : +/- position offset
  • IconsAsBackground : Determines if the icons m ust be used as background images: True or False
  • IconsMaximize : Determines if the icon image must be maximized or not: True or False
  • IconsTextGap : Distance in pixels between icon and text
  • IconsFixedTextOffset : True or False → IconsTextGap
  • ResizeableColumns : Determines if columns can be resized using the mouse: True or False → default True
  • OCStyle :  Open/Close sybols style (1=Box, 2=Arrows)
  • SelectGroups : Enable/Disable group selection (default = True)
     

 

Additional supported actions 

  • OnChange : Triggered when something changes
  • OnSelect : Triggered when an item has been selected
  • OnUnselect / OnDeselect : Triggere when an item has been deselected
