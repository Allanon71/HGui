BUTTON GADGET METHODS

  • gadObj:Aligns(align, refresh)
    Set the button's alignments for each states separately if you pass a tablewith single states fields (normal, pushed, disabled, hilighted), or, if    you pass an alignment constant, it will be applied to all states. Constants are defined here.
    - align: A table like this:
    { normal = #HGUI_ALIGN_LEFT, pushed = #HGUI_ALIGN_RIGHT, etcâ€¦ }
    or an alignment constant that will be applied to all button states.
    - refresh: True to refresh the gadget when finished
     
  • gadObj:setCaptions(texts, refresh)
    Set the button's caption for each states separately if you pass a table with single states fields (normal, pushed, disabled, hilighted), or, if  pass a string, it will sets the given caption on all states.
    If you pass a table of string with a numeric index than it will be interpreted as a multiline string to apply to all states.
    - table: A table like has been described above.
    - refresh: True to refresh the gadget when finished
