SWITCH GADGET METHODS

  • result = gadObj:getValue()
    Returns the switch value (0 = up or 1 = down)
     
  • gadObj:setValue(value)
    Sets the switch value (0 = up or 1 = down)
    - value : 0 = up or 1 = down
     
  • result = gadObj:isDown()
    Returns True if the switch is pressed
     
  • result = gadObj:isUp()
    Returns True if the switch is not pressed
     
  • result = gadObj:setCaptions(texts)
    Set the switch captions using a table with 2 fields as follow:
    { normal = <text>, pushed = <text> }
    Text can be a string or a table of strings for multiline texts.
    - texts : A table used to change the switch text for each state:
      normal : sets the text for normal, hilightedand disabled texts
      pushed : sets the text for pushed, normal-pushed, hilighted-pushed and disabled-pushed states

 

Additional arguments accepted by :New() method

  • Value : 1 = Switch ON (Down), 0 = Switch OFF (Up)

 

Additional actions accepted by :New() method

  • OnPushed : Triggered when switch has been pushed and its state changed
