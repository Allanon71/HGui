GROUP GADGET METHODS

  • gadList = gadObj:getValues()
    Returns a list (table) of included gadgets and their values, each entry has this format:
      { index = n, name = s, value = n, gadget = <gadobject> }
    - gadList : A table composed of records as explained above 

 

Additional arguments accepted by :New() method

  • Outer_Box : BOX class options to customize the outer box
  • Inner_Box : BOX class options to customize the inner box
  • Header : LABEL class options to customize the group header
  • Group_LabelSide : Specify the header label position : 
    #HGUI_ALIGN_TOP, #HGUI_BOTTOM, #HGUI_LEFT, #HGUI_RIGHT
  • Group_LabelGap : Value used to increase/deacrease the header size
  • GroupType : Specify the group type : 
    #HGUI_GTYPE_GENERIC, #HGUI_GTYPE _CHECK, #HGUI_GTYPE_RADIO

 

NOTES

If caption is not specified or it is an empty string the label gadget will be replaced by an empty box with the minimum possible size.
