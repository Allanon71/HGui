HGui.Window:ProgressUpdate()

winObj:ProgressUpdate(params)

Updates an existing progress bar window with what you have specified in params.

INPUT

  • params : A table with one or more fields to update:
    • value : The new value to set as current value.
    • message : Text message to update.
