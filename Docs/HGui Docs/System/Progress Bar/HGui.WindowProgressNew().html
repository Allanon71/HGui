HGui.Window:ProgressNew()

winObj = HGui.Window:ProgressNew(params)

Create a new window with a progress bar.

INPUT

  • params : A table that defines the window, see :New() method for details. Also these additional tags are supported:
    • valueMin : Minimum value for the progress bar, default = 1
    • valueMax : Maximum value for the progress bar, default = 100
    • value : Current value for the progress bar, default = 1
    • message : Text message to show, default is an empty string.
    • cursor : Custom cursor (optional)
    • cursorText : Text to place into the cursor (default is "{value}/{rangeend}"), it supports the following tags (you can use more than one tag):
      • {value} : Value
      • {value.##} : Value rounded to  2 decimal numbers
      • {intvalue} : Integer value
      • {rangstart} : The value specified in valueMin
      • {rangeend} : The value specified in valueMax
      • {perc} : The current value percentual
    • size : A table with the w and h fields that represent the InputBox size in pixels, default is { w = 300, h = 100 }

OUTPUT

  • winObj : Returns the pointer to the progress window.

NOTES
You can close this window by calling the :Free() method as for normal windows.
