HGui.SetQuitOnLastWindow()

HGui.SetQuitOnLastWindow(value)

Switch the auto-quit feature on last-window-close to ON or OFF.

INPUT

  • value : TRUE to activate, otherwise FALSE

OUTPUT

NOTES
If this feature is active, when the last opened window is closed the application will be automatically ended. If the feature is not active, instead, the application will continue to run without any opened window.
