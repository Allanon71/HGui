HGui.Window:WaiterUpdate()

waiterBox:WaiterUpdate(text, fg, bg)

Updates the waiterBox changing its text and optionally the foreground and/or the background colors.

INPUT

  • text : A new string to display in the box
  • fg : Optional foreground color
  • bg : Optional background color

OUTPUT

  • winObj : The waiterBox window's object
