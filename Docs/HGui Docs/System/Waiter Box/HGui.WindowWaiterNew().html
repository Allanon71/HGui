HGui.Window:WaiterNew()

winObj = HGui.Window:WaiterNew(params)

Shows an animated wait window with an animation as a busy state indicator.

INPUT

  • params : A table with a list of parameters to customize the waiterBox
    • text : Text message to show into the box
    • freq : Animation frequency in ms (default = 100)
    • size : A table with the window size specified in the fields w and h (default = 400x100)
    • fg : Animated bar foreground color
    • bg : Animated bar background color
    • round : Animated bar segment's rounding level (default = 0)
    • segments : Animated bar number of segments (default = 16)

OUTPUT

  • winObj : The waiterBox window's object
