Tests & Examples

Here is a list of included tests & examples, you can find them inside the /examples/HGui folder.

Every script is commented so you can have an idea how it works.

 

THEME RELATED SCRIPTS
These scripts can be used to learn how to set gadget skins and how to create gadgets.

SHOWCASE
* 000 - Showcase.hws
  This is a complex program, check its code when you have learned a bit of HGui, run it to get an idea about HGui capabilities.
