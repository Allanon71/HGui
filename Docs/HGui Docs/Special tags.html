Special tags

There are soime special tags you can embed at the beginning of the text to temporarily change how the text is rendered, the speciafied changes are applied only to the line where they has been specified.

At this time the following tags can be used safely with LABEL and BUTTON gadgets:

{:align-l:}      		Align the line to the left
{:align-r:}      		Align the line to the right
{:font++:}          	Increase the font size for the line
{:font--:}          	Decrease the font size for the line
{:font=nnn:}        	Set the font line size with the given nnn value, nnn must be 3 characters long
{:bgcolor=aarrggbbz:}   Render the background of the full line with the given hex color, z is the background box round level (0-9)
{:bbcolor=aarrggbbz:}   Same as above but the background is rendered only under the text.
