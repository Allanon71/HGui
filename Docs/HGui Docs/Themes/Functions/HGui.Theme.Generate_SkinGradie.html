HGui.Theme.Generate_SkinGradient()

 

skinDef = HGui.Theme.Generate_SkinGradient(color1, color2)

Generate a simple skin composed by a single layer with 2 colors gradient.

INPUT

  • color1 : ARGB color 1 used for the gradient
  • color2 : ARGB color 2 used for the gradient

OUTPUT

  • skinDef : A table with the skin definition
