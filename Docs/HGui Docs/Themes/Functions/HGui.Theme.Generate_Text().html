HGui.Theme.Generate_Text()

skinDef = HGui.Theme.Generate_Text(size, style, color, align, edge, shadow, margins)

Generate a simple a CText object you can used to skin gadgets.

INPUT

  • size : Font size in pixels
  • style : Font style using Hollywood inbuilt constants
  • color : ARGB text color
  • align : Text alignment using one or more of the following constants:
	#HGUI_ALIGN_LEFT
	#HGUI_ALIGN_RIGHT
	#HGUI_ALIGN_TOP
	#HGUI_ALIGN_BOTTOM
	#HGUI_ALIGN_HCENTER
	#HGUI_ALIGN_VCENTER
	#HGUI_ALIGN_CENTER
  • edge : Set to True to apply a raised effect
  • shadow : Set to True to apply a shadow effect
  • margins : An optional table with the text margins in pixels with the following items:
	top
	bottom
	left
	right

OUTPUT

  • CTextObj : A CText object
