HGui.Theme.Generate_SkinBevel()

skinDef = HGui.Theme.Generate_SkinBevel(type, fx, height, light, dark, middle)

Generate a simple skin composed by a single layer with a bevel box effect.

INPUT

  • type : Bevel effect type, one of the following
    • GFX.BevelType_Standard
    • GFX.BevelType_Sunken
  • fx : Bevel box effect, one of the following
    • GFX.BevelFx_Raised
    • GFX.BevelFx_Recessed
    • GFX.BevelFx_Flat
  • height : Bevel effect height in pixels
  • light : ARGB color to use for the hilighted edges
  • dark : ARGB color to use for the dark edges
  • middle : ARGB color to use for the fill color

OUTPUT

  • skinDef : A table with the skin definition
