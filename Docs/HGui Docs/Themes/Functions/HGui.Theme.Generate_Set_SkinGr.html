HGui.Theme.Generate_Set_SkinGradient()

states = HGui.Theme.Generate_Set_SkinGradient(color1, color2)

Generate a set of skins, one for each gadget state, with a 2 colors gradient in two layers, one with the gradient and one with a bevel box. Colors are modified for each generated state as described below:

  • nskin : normal state, color 1 is brighten by 10 units, color 2 is darken by 10 units.
  • hskin : hilighted state, color1 is brighten by 20 units, color 2 is brighten by 20 units.
  • pskin : pushed state, as hskinDef but colors are inverted
  • dskin : disabled state, color 1 is darken by 20 units, color 2 is brighten by 20 units, gradient is not applied

INPUT

  • color1 : color used for the bright edges and as the starting gradient color
  • color2 : color used for the dark edges and as the ending gradient color

OUTPUT

  • states : A set of skinDef tables with the following items
    • nskin (normal state skin)
    • hskin (hilighted state skin)
    • pskin (pushed state skin)
    • dskin (disabled state skin)

NOTES

This function is particularly suited for generating button skins
