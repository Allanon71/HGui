HGui.Theme.Generate_SkinDoubleGradient()

skinDef = HGui.Theme.Generate_SkinDoubleGradient(color1, color2, h)

Generate a simple skin composed by a two layers with inverted gradient as fill color.

INPUT

  • color1 : ARGB color 1 used for the gradient
  • color2 : ARGB color 2 used for the gradient
  • h : optional border in pixels between the inner and the outer boxes, default = 2

OUTPUT

  • skinDef : A table with the skin definition
