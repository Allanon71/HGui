HGui.Theme.SetGlobalTheme()

result = HGui.Theme.SetGlobalTheme(gadgetType, Theme)

Apply the given Theme to all gadget classes specified by gadgetType only if the gadget's theme has not been customized at creation time.

INPUT

  • gadgetType : Can be any of the gadget constant specified hereor you can use the #ALL constant to apply the theme to all the supported gadgets.
  • Theme : A complete theme table to apply to the gadgets as described here

OUTPUT

  • result : True if the operation was completed without errors otherwise False

NOTES

Custom skins will not be changed.  A custom skin is a skin that has been set using the gadget's method :setTheme()
 This function is in are build/fix state
