HGui.Theme.Generate_SkinColor()

 

skinDef = HGui.Theme.Generate_SkinColor(color)

Generate a simple skin composed by a single layer with a plain fill color.

INPUT

  • color : ARGB Color used to fill the area layer

OUTPUT
