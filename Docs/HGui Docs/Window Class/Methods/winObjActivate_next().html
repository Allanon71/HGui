winObj:Activate_next()

result = winObj:Activate_Next(remove_this)

Activate the next window on the windows sorting stack. If remove_this is True then this window object will be removed from the sorting stack.

INPUT

  • remove_this : True to remove from the stack the current window

OUTPUT

  • result : True if completed without errors

NOTES

Mostly used internally to handle the window's focus
