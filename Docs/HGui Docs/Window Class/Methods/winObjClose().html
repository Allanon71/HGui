winObj:Close()

winObj:Close()

Close this window if it's currently opened.

INPUT

OUTPUT

  • result : False if the window is already closed.

NOTES

Along with the closing operation all event handlers will be disabled. 

As soon as the window is closed another one will be activated following the current windows order stack.

Hidden windows cannot be closed directly.
