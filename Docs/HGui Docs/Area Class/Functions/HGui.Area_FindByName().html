HGui.Area_FindByName()

areaObj, index = HGui.Area_FindByName(area_name)

Searches the area named area_name and returns the found object and its index in HGui.Areas table, otherwise returns False in areaObj result variable.

INPUT

  • area_name : The name of the area to search for

OUTPUT

  • areaObj : False if nothing was found, otherwise the areaObj
  • index : Area index of HGui.Areas table, where it has been stored.
