HGui.Area:New()

areaObj = HGui.Area:New(options, object, userdata)

Builds a new graphic area. 

A graphic area is an area associated with the given window (or, if not specified, the currently active one will be selected) that can be skinned and can react to the mouse events triggering callback functions.
All parameters are optional, default values are defined in the object data structure.

INPUT

  • options : A table of all default values you want to override, like position, size, callback functions, etcâ€¦
  • object : An HGui object attached to this area
  • userdata : Any type of data you want to get back by the callback functions

OUTPUT

  • areaObj : A new area object.
