HGui.ShowSplashScreen()

HGui.ShowSplashScreen(imageFile, size, setDisplaySize)

Open a window on display 1 to show the loading progress of an application.

You have to use it before any HGui window!

INPUT

  • imageFile : [STR] Image file to show in the splash screen
  • size : [VAR] Can be a string or a table with the following meanings:
    • String : Set to image to use the original image's size
    • Table : Pass a table with the fields w and h to specify the splash image's width and height
  • setDisplaySize : [BLN] If TRUE the display's size will get the image's size

OUTPUT

NOTES
This function is used to show a splash screen during the startup of your applications, the image will fade in and you can also print a text and show a progress bar while loading your app.
The animation is a moving bar on the bottom of the image with an optional text message.

 -- internal notes ---
 All informations about the splash screen are stored in the table HGui.Splash, it has the following fields:

   .w          Splash screen width
   .h          Splash screen height
   .image      Splash screen image brush (removed as soon as it is displayed)

The following fields, instead, are hardcoded

   .animWidth  Animation width
   .animStep   Animation step
   .animColor  Animation color
   .animX      Animation negative width (for left entering fx)
