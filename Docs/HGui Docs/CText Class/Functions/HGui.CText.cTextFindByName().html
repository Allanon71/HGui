HGui.CText.cTextFindByName()

ctextObj, index = HGui.CText.cTextFindByName(name)

Search a cText object by its name.

INPUT

  • name : Name of the cText object we are looking for

OUTPUT

  • ctextObj : cText object or False if nothing was found.
  • index : cText object index in HGui.CTexts table.
