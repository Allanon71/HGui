Data Structure

Position   		[Table] Area position, must have the following fields:

Name			[String] Object Name
First_Line		[Number] First line to display
Caption         [Table] Object's text contents
Wordwrap		[Boolean] Text wordwrap
Window			[Table] Parent window object
Interline		[Number] Additional interline in pixels
Angle			[Number] Text angle
Encoding		[Number] Text Encoding (Hollywood's constants)
Vert_Adjust		[Number] Vertical adjustment in pixels
Font			[Table] Font definition
	.Name 		[Number/String] Font Name (Constant or font name)
	.Size		[Number] Font Size
	.Style		[Number] Font Style (Hollywood's constants)
	.Color		[Number] Font Color
	.Align		[Number] Font Alignment (HGui's constants)
Effects			[Table] Font effects
	.Edge		[Table] Edge effect definition
		.Type	[Number] Effect type
		.Size	[Number] Edge height
		.Color1 [Number] Color 1
		.Color2 [Number] Color 2
	.Shadow		[Table] Shadow effect definition
		.Type 	[Number] Shadow type (Hollywood's constant)
		.Distance [Number] Shadow distance
		.Color 	[Number] Shadow color
Margins 		[Table] Text margins
	.Top		[Number] Top margin
	.Bottom		[Number] Bottom margin
	.Left		[Number] Left margin
	.Right		[Number] Right margin
Area 			[Table] Associated Area Object
