ctextObj:Render()

result = CTextObj:Render(area, clipArea)

Renders the CText object in its parent window, using its area object.
You can override the area using a custom area, object furthermore you can apply a cliparea to delimit the rendering area.

INPUT

  • area : A table with at least the following fields:
    • position : Area position table
      • x : Horizontal position
      • y : Vertical position
    • size : Area size table
      • w : Width
      • h : Height
  • clipArea : A table with the following fields
    • x : Horizontal position
    • y : Vertical position
    • w : Width
    • h : Height

OUTPUT

  • result : True if there was no error otherwise False

NOTES

This function needs improvements to render only the visible lines,  actually it renders all the text lines from the first to the last.
